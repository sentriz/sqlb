# sqlb

[![go reference](https://pkg.go.dev/badge/go.senan.xyz/sqlb.svg)](https://pkg.go.dev/go.senan.xyz/sqlb)

```sh
go get go.senan.xyz/sqlb                   # library
go get -tool go.senan.xyz/sqlb/cmd/sqlbgen # tool for codegen
```
